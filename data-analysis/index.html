데이터 엔티티 관계도 (ERD)

eCross 로그 시스템의 계층적 구조와 논리적 인과 관계

설명력(XAI) 확보의 핵심: 하위 단계(Message/Body)의 이상 징후가 상위 단계(Transaction)의 실패로 전이되는 인과 관계를 계층 구조로 추적합니다.

1 : 1

EC_MESSAGE_BODY

PK/FK MESSAGE_ID

Layer 1

EC_TRANSACTION_MONITORING

비즈니스 트랜잭션의 전체 수명 주기를 관리하는 최상위 마스터 데이터

SQL Schema

PKTRANSACTION_IDVARCHAR2(80)
—INTERFACE_IDVARCHAR2(30)
—STATUSVARCHAR2(10)
—PROCESS_TIMENUMBER
—START_TIMETIMESTAMP
—END_TIMETIMESTAMP
—RETRY_COUNTNUMBER

PROCESS_TIME 분포 시뮬레이션

시간대별 소요시간(ms) — 이상치는 ● 빨간 점

확인 사항

PROCESS_TIME 분포 확인 — 전체 지연(Latency) 판단 기준
STATUS ≠ 'S' 인 모든 케이스를 이상 후보로 간주

전처리 TIP

주기성 학습

START_TIME에서 시간대(0–23시)·요일 특징을 추출해 업무/비업무 시간 패턴을 분리 학습

로그 스케일링

처리 시간 편차가 크므로 Log Scaling으로 분포를 정규화해 모델 수렴 속도를 향상

Layer 2

EC_PROCESS_MONITORING

트랜잭션 내부의 세부 작업 단위(어댑터 실행 등)를 기록하는 워크플로우 데이터

SQL Schema

PKPROCESS_IDVARCHAR2(36)
FKTRANSACTION_IDVARCHAR2(80)
—DEPEND_PROCESS_IDVARCHAR2(36)
—ADAPTER_TYPEVARCHAR2(30)
—SUCCESS_COUNTNUMBER
—ERROR_COUNTNUMBER
—START_TIMETIMESTAMP
—END_TIMETIMESTAMP

어댑터 구간별 평균 처리 시간

병목 구간은 ■ 주황색으로 강조

확인 사항

DEPEND_PROCESS_ID 기반 프로세스 간 선후 관계 지연 추적
특정 ADAPTER_TYPE에서 반복 발생하는 지연 패턴 식별

전처리 TIP

그래프 구조화 (DAG)

선후 관계를 DAG로 표현하고, 단계 간 시간을 에지(Edge) 가중치로 활용해 비정상 실행 경로를 탐지

Layer 3

EC_MESSAGE_MONITORING

프로세스 수행 시 발생하는 실제 데이터(메시지)의 물리적 흐름 기록

SQL Schema

PKMESSAGE_IDVARCHAR2(36)
FKPROCESS_IDVARCHAR2(36)
—DIRECTIONVARCHAR2(3)
—DATA_SIZENUMBER
—RESPONSE_CODEVARCHAR2(10)
—PROCESS_DTTIMESTAMP

DATA_SIZE 분포 — 이상 탐지

3-Sigma 범위 초과 데이터는 ● 이상 후보

확인 사항

RESPONSE_CODE = '0000'(성공) 임에도 DATA_SIZE = 0 인 케이스 탐지
평소보다 비정상적으로 큰 DATA_SIZE — 과부하 요청 여부 확인

전처리 TIP

Z-Score 활용

DATA_SIZE의 평균·표준편차로 3-Sigma 범위를 산출해 데이터 유실 및 과부하 요청을 피처링

Layer 4

EC_MESSAGE_BODY

실제 송수신된 비정형 데이터 전문(Payload)을 보관하는 최종 단계

SQL Schema

PK/FKMESSAGE_IDVARCHAR2(36)
—MESSAGE_INCLOB
—MESSAGE_OUTCLOB
—MESSAGE_BODYCLOB

AutoEncoder 재구성 손실 비율

Reconstruction Loss 기준 정상/이상 분류

확인 사항

JSON/XML 스키마 키(Key) 구성 변화 탐지 — 키 추가·누락 시 이상
비즈니스 로직 불일치 및 예상 외 필드 값 범위 이탈

전처리 TIP

Flattening

중첩 JSON을 Key-Value 평면 테이블로 변환 후 One-Hot·정규화 처리

AutoEncoder

평탄화 벡터를 입력해 재구성 손실이 높은 패킷을 내용 결함 이상으로 분류
